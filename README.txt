=== INSTRUCTIONS D'INSTALLATION ===

1. Décompressez ce dossier.
2. Copiez votre fichier 'evolution_indicateurs_cles_ac.json' dans le dossier 'public'.
   (C'est indispensable pour que les données s'affichent !)
3. Ouvrez un terminal dans ce dossier.
4. Lancez la commande : npm install
5. Assurez-vous que votre base de données MySQL est lancée (WAMP/XAMPP).
6. Lancez le serveur : node server.js
7. Allez sur http://localhost:3000
